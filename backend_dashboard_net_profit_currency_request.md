# طلب دعم صافي الأرباح بالعملات المحلية والدولية

مرحبًا فريق الباك-إند،

أضفنا بطاقات جديدة في لوحة التحكم الإلكترونية لعرض **صافي الأرباح** مقسّمًا حسب العملات الثلاثة الأكثر استخدامًا لدينا: الجنيه المصري، الريال السعودي، والدولار الأمريكي. الواجهة الأمامية أصبحت جاهزة لعرض هذه القيم، لكنها تعتمد على توفر الحقول التالية في استجابة واجهة `GET /api/Dashboard/overview`:

## الحقول المطلوبة داخل `metrics`
- `netProfitEgp` مع `netProfitEgpCurrencyCode` (يفضل أن تكون القيمة `EGP`).
- `netProfitSar` مع `netProfitSarCurrencyCode` (يفضل أن تكون القيمة `SAR`).
- `netProfitUsd` مع `netProfitUsdCurrencyCode` (يفضل أن تكون القيمة `USD`).

> **ملاحظة:** ما زلنا نستخدم `netProfit` و`netProfitCurrencyCode` للعرض المجمع، لذا يرجى الإبقاء على هذه القيم أيضًا.

## سلوك الاستجابة المقترح
- القيم يمكن أن تكون رقمية أو نصية جاهزة للعرض. عند غياب القيمة يمكن إرجاع `null`.
- إذا كانت هناك فترة زمنية أو دور محدد يؤثر على الحساب، يرجى تطبيق نفس المنطق المستخدم لبقية مؤشرات `metrics` حتى تتماشى النتائج مع بطاقات "المعاملات المالية".

## مثال مختصر للاستجابة
```json
{
  "role": "Admin",
  "metrics": {
    "netProfit": 125000.50,
    "netProfitCurrencyCode": "USD",
    "netProfitEgp": 3000000,
    "netProfitEgpCurrencyCode": "EGP",
    "netProfitSar": 470000.75,
    "netProfitSarCurrencyCode": "SAR",
    "netProfitUsd": 125000.5,
    "netProfitUsdCurrencyCode": "USD"
  }
}
```

إذا احتجتم أي توضيحات إضافية حول كيفية احتساب القيم لكل عملة أو مصادر البيانات المتاحة، يسعدنا التنسيق معكم. شكرًا مقدمًا على تعاونكم.

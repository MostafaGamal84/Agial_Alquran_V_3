# طلب إنشاء واجهة برمجية لجلب تفاصيل الحلقة مع الطلاب

مرحبًا فريق الباك-إند،

أثناء العمل على شاشة إنشاء التقارير التي تعتمد على اختيار الحلقة والطالب، نستخدم حاليًا نداء HTTP:
```
GET /api/Circle/Get?id={circleId}
```
لكن هذه الواجهة غير متوفرة حاليًا على الخادم، مما يمنع الواجهة الأمامية من تحميل طلاب الحلقة بعد اختيارها.

## المطلوب
توفير نقطة نهاية تُعيد تفاصيل الحلقة المحددة بـ `id`، مع تضمين المدرس، المشرفين، وقائمة الطلاب المرتبطين بها.

### المقترح
- **المسار**: `GET /api/Circle/Get`
- **المعلمات**: `id` (رقم معرف الحلقة)
- **الاستجابة المتوقعة (JSON)**:
  ```json
  {
    "id": 27,
    "name": "اسم الحلقة",
    "teacherId": 123,
    "teacher": { "id": 123, "fullName": "اسم المعلم" },
    "managers": [
      { "managerId": 45, "manager": { "id": 45, "fullName": "اسم المشرف" } }
    ],
    "students": [
      { "studentId": 1001, "student": { "id": 1001, "fullName": "اسم الطالب" } },
      { "studentId": 1002, "student": { "id": 1002, "fullName": "اسم الطالب" } }
    ]
  }
  ```

### ملاحظات إضافية
- هيكل الحقول يتوافق مع الواجهات المستخدمة حاليًا في الواجهة الأمامية (`CircleDto` وحقول `students`/`studentId`)، لذلك يمكن إعادة استخدامه مباشرة دون تعديلات إضافية على الواجهة.
- إذا كان هناك مسار بديل متاح حاليًا يعيد البيانات نفسها، يرجى مشاركته لتحديث الاستدعاء بدلًا من إضافة مسار جديد.
